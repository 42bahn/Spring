# Spring MVC (Model / View / Controller)

- Spring 프레임워크에서 제공하는 웹 모듈이다.
- 백엔드 프로그래밍의 기본 프레임워크
- MVC 는 Model-View-Controller 의 약자로, 기본 시스템 모듈을 MVC 로 나누어 구현되어있다.

    - Model : Data 디자인

    - View : 실제로 렌더링되어 보이는 페이지

    - Controller : 클라이언트의 요청과 응답 처리
